API Locadora
=================
# Introduction 
Api that provides services for the locadora application used to control movie rental.

9. FluentAssertions 6.6.0
10. Microsoft.NET.Test.Sdk 16.11.0
11. Moq 4.17.2
12. Xunit 2.4.1
13. coverlet.msbuild 3.1.2
